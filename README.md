- 진하게 : strong</br>
strong 태그는 해당 콘텐츠의 중요성이나 심각함, 긴급함 등을 강조할 때 사용합니다.</br>
- 밑줄 : u </br>
u 태그는 철자가 틀린 단어나 중국어의 고유 명사처럼 문체상 일반적인 텍스트와는 달라야만 하는 텍스트를 표현할 때 사용합니다.</br>
- 제목 : h1 ~ h6</br>
HTML 문서에서 제목(heading)을 정의할 때 사용합니다.</br>
